# Data Processing Project

## Overview
This is a data processing project that handles data cleaning and analysis.

## Project Structure
- `main.py` - Main application entry point
- `data_processing.py` - Data processing utilities
- `pre_process.py` - Data preprocessing functions
- `csv_data/` - Contains CSV files for processing

## Setup
1. Activate the virtual environment
2. Install dependencies from `requirements.txt`
3. Run `python main.py`

## Features
- Data preprocessing and cleaning
- Data analysis and reporting
- Support for multiple CSV formats

## Requirements
See `.env` file for environment configuration
